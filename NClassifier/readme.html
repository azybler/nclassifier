Welcome to NClassifier

NClassifier is a .NET based text classification classification engine which supports, among others, Bayesian classification and text summarization.  It is a port of the popular Classifier4J tool by Nick Lothian.

Latest Version

The latest version of NClassifier can be found on SourceForge here:
http://nclassifier.sourceforge.net

Documentation

Right now there is no documentation except that which is in the source code.  Source code comments will be expanded on for an eventual NDoc run.

I would recommend looking at the source in the NUnit tests to better see how to use the NClassifier engine.

I would also recommend visiting Classifier4J's website here:
http://classifier4j.sourceforge.net/
